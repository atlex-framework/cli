# @atlex/cli

> Atlex command-line interface (`atlex new`, generators, `migrate`, `queue:work`, `serve`, …).

[![npm version](https://img.shields.io/npm/v/@atlex/cli.svg?style=flat-square&color=7c3aed)](https://www.npmjs.com/package/@atlex/cli)
[![TypeScript](https://img.shields.io/badge/TypeScript-strict-3178C6?style=flat-square)](https://www.typescriptlang.org/)

[![Buy Me A Coffee](https://img.shields.io/badge/Buy%20Me%20A%20Coffee-Support-yellow?style=flat-square&logo=buy-me-a-coffee)](https://buymeacoffee.com/khamazaspyan)

## Installation

```bash
npm install --save-dev @atlex/cli
```

## Quick start

```bash
npx atlex --help
npx atlex new my-app
npx atlex migrate
```

## Features

- Project scaffolding (`atlex new`)
- Code generators (`make:*`)
- Database, queue, schedule, and config commands
